# Netflix 1行スクロール

Chrome拡張機能 — マウスホイール1ノッチで、Netflixの行を **ぴったり1行分**(超重要) スナップスクロール。

Netflixのブラウズ画面でホイールを回すたびに、カーソル下の行が **完璧に揃った状態** で次の行へ移動。もう「カーソルを作品(アイテム)に合わせていたのにスクロールしたらズレた😢」は起きません！！！


---

## 特徴

- **絶望的にいらない** - ほしい人は多分おちゃだけです。
- **スナップスクロール** — ホイール1ノッチ = 正確に1行分!!
- **スムーズアニメーション** — easeOutCubicで滑らかに移動（180ms）
- **アニメーション中も連続対応** — 立て続けにスクロールしても目標位置から計算
- **動画再生中は無干渉** — `/watch` ページでは何もしない
- **横スクロール・修飾キーは標準動作** — Ctrl/Shift/Alt併用時や横スクロールは邪魔しない
- **クラス名変更に耐性** — Netflixのクラス名が変わっても複数セレクタでフォールバック

---

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の **デベロッパーモード** をオン
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. このフォルダを選択
5. Netflixのタブをリロード

---

## 設定

`content.js` 先頭の `SETTINGS` を変更し、拡張機能をリロードすると挙動を調整できます。

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| `duration` | `180` | アニメーション時間（ms）。`0` で瞬間移動 |
| `offset` | `0` | 行の上端を画面のどこに合わせるか（px）。固定ヘッダーに被るなら `60` 等 |
| `epsilon` | `2` | 行境界の判定誤差（px） |

---

## 仕様

- ホイールイベントを横取りし、`.lolomoRow` / `.rowContainer` / `[data-list-context] .row` の位置を測って隣の行境界へスクロール
- Netflix側のクラス名が変わって行が検出できない場合、標準スクロールにフォールバック
- `passive: false` + `capture: true` で確実にイベントを横取り
