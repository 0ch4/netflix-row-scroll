# 1行スクロール (Netflix / YouTube)

Chrome拡張機能 — マウスホイール1ノッチで、**ぴったり1行分**(超重要) スナップスクロール。

Netflix・YouTubeのブラウズ画面でホイールを回すたびに、カーソル下の行が **完璧に揃った状態** で次の行へ移動。もう「カーソルを作品(アイテム)に合わせていたのにスクロールしたらズレた😢」は起きません！！！

> YouTubeはサムネがグリッド状（1行に複数個）なので、各アイテムの上端をクラスタリングして「行」を割り出してスナップします。ホーム/登録チャンネル/検索結果どれでもOK。


---

## 特徴

- **絶望的にいらない** - ほしい人は多分おちゃだけです。
- **スナップスクロール** — ホイール1ノッチ = 正確に1行分!!
- **スムーズアニメーション** — easeOutCubicで滑らかに移動（180ms）
- **アニメーション中も連続対応** — 立て続けにスクロールしても目標位置から計算
- **動画再生中は無干渉** — `/watch` ページでは何もしない
- **横スクロール・修飾キーは標準動作** — Ctrl/Shift/Alt併用時や横スクロールは邪魔しない
- **Netflix / YouTube 両対応** — グリッドもリストもアイテムのクラスタリングで1行判定
- **クラス名変更に耐性** — クラス名が変わっても複数セレクタでフォールバック

---

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の **デベロッパーモード** をオン
3. **パッケージ化されていない拡張機能を読み込む** をクリック
4. このフォルダを選択
5. Netflix / YouTube のタブをリロード

---

## 設定

`content.js` 先頭の `SETTINGS` を変更し、拡張機能をリロードすると挙動を調整できます。

| 項目 | デフォルト | 説明 |
|------|-----------|------|
| `duration` | `180` | アニメーション時間（ms）。`0` で瞬間移動 |
| `offset` | `"auto"` | 行の上端を画面のどこに合わせるか。`"auto"`=サイトごと自動（YouTubeはヘッダー分下げる / Netflixは0）。数値でpx指定も可 |
| `epsilon` | `2` | 行境界の判定誤差（px） |
| `cluster` | `6` | この差（px）以内のアイテム上端を「同じ行」とみなす |

---

## 仕様

- ホイールイベントを横取りし、アイテムの上端Y座標を測定 → 近いものをクラスタリングして「行の境界」を算出 → 隣の境界へスナップ
- **Netflix**: `.lolomoRow` / `.rowContainer` / `[data-list-context] .row`
- **YouTube**: `ytd-rich-item-renderer` / `ytd-video-renderer` / `ytd-grid-video-renderer` / `ytd-rich-grid-row`
- 動画再生中（`/watch`）は無干渉
- 行が検出できない場合、標準スクロールにフォールバック
- `passive: false` + `capture: true` で確実にイベントを横取り
